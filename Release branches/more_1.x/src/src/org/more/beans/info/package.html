more.beans的核心信息包，可以通过实现{@link org.more.beans.BeanResource}接口来决定如何提供BeanDefinition对象。
有了BeanDefinition整个more.beans就可以工作起来了。

Prop
属性名 属性类型 默认值 说明
id String null 唯一的Bean ID。
propType String null 该类型应该可以通过BeanFactory.getBeanClassLoader().loadClass()获取。

 

BeanDefinition
属性名 属性类型 默认值 说明
id String null 唯一的Bean ID。
propType String null 该类型应该可以通过BeanFactory.getBeanClassLoader().loadClass()获取。
name String null 在一个beanFactory中的唯一名称。
lazyInit boolean true 是否延迟初始化这个bean，只有当bean是单态模式下才生效。
description String null bean描述信息。
iocType {@link org.more.beans.info.IocTypeEnum IocTypeEnum} IocTypeEnum.Ioc 依赖注入方式，{@link org.more.beans.core.injection.InjectionFactory InjectionFactory}依赖这个属性，默认值为Ioc。
exportIocRefBean String null 依赖注入方式为export时导出的属性注入器bean名。
propertys {@link org.more.beans.info.BeanProperty BeanProperty}[] null bean中注册的属性这些属性需要依赖注入。
createType {@link org.more.beans.info.CreateTypeEnum CreateTypeEnum} {@link org.more.beans.info.CreateTypeEnum#New CreateTypeEnum.New} 创建方式，默认为New。
constructorParams {@link org.more.beans.info.BeanProperty BeanProperty}[] null 配置的构造方法参数。当创建方式为工厂方式时，构造方法将失效，一切创建代码委托给工厂方法。
factoryRefBean String null 使用工厂方式创建时的工厂bean名称。
factoryIsStaticMethod boolean false 调用工厂类的方法是否为一个静态方法。
factoryMethodName String null 调用工厂类的方法名
factoryMethodParams {@link org.more.beans.info.BeanProperty BeanProperty}[] null 调用工厂类的方法时需要传递的参数，参数依照数组中元素顺序。
aopFilterRefBean String[] null AOP过滤器bean，CreateTypeEnum如果为{@link org.more.beans.info.CreateTypeEnum#Factory Factory}方式则AOP使用{@link org.more.core.classcode.ClassEngine.BuilderMode#Propxy Propxy}代理方式创建， 如果是{@link org.more.beans.info.CreateTypeEnum#New New}方式创建则创建模式使用{@link org.more.core.classcode.ClassEngine.BuilderMode#Super Super}方式创建。
implImplInterface {@link org.more.beans.info.BeanInterface BeanInterface}[] null 要附加实现的接口
isSingleton boolean false 是否为单态模式

 

BeanConstructor
属性名 属性类型 默认值 说明
id String null 唯一的Bean ID。
propType String null 该类型应该可以通过BeanFactory.getBeanClassLoader().loadClass()获取。
constructorParams {@link org.more.beans.info.BeanProperty BeanProperty}[] null 构造方法参数表。

 

BeanInterface
属性名 属性类型 默认值 说明
id String null 唯一的Bean ID。
propType String null 该类型应该可以通过BeanFactory.getBeanClassLoader().loadClass()获取。
typeRefBean String null 接口类型引用Bean名，配置了typeRefBean属性则可以不配置type属性。
implDelegateRefBean String null 附加接口实现时接口处理委托bean名。

 

BeanProperty
属性名 属性类型 默认值 说明
id String null 唯一的Bean ID。
propType String null 该类型应该可以通过BeanFactory.getBeanClassLoader().loadClass()获取。
name String null 属性名，对于构造方法参数配置该值无效。
value {@link org.more.beans.info.BeanProp BeanProp} null 属性值。
