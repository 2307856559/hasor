submit系统v2.0 
说明： 
 submit系统v2.0。submit系统当前版本重写了系统底层架构并且不在必须依赖web服务器。submit系统可以应用在普通桌面程序。 
 控制台程序。如果你想在程序中使用请求转发器那么就可以使用submit2.0组建。submit2.0组建是一个完全独立的mvc框架。它的
 工作是对一个action请求反复拦截进行处理，而拦截器的编写完全由用户自己开发。作为action只需要编写任意方法。方法中只能
 有一个ActionMethodEvent类型的参数既可。根据外壳接口实现不同submit2.0可以自定义action源和过滤器源。这些扩展在casing软件包
 中会有全面介绍。
 不仅如此，submit2.0还有灵活的支撑环境，甚至你也可以简单的编写一个它的支撑环境，所谓submit2.0支撑环境就是在某个项目
 中如何使用submit2.0的转发功能类。由于submit2.0不在依赖web因此对于web需要一个支撑环境。submit2.0在编写时候提供了
 filter方式、servlet方式、JSP标签方式。通过actionjs插件目前submit2.0也支持了javascript无刷新方式调用转发功能。
 submit2.0小巧，它只有12个核心类构成主要结构。并配合外壳扩展来完成自己的集成丰富。
