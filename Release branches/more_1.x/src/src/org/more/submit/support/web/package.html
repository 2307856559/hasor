submit系统v3.0的web支撑环境。
目前的支撑环境已经可以支持Filter、Servlet、JSP标签三种内置方式调用action方法。
Filter配置：
	Class      : org.more.submit.support.web.TopFilter
	buildClass : org.more.submit.casing.more.WebMoreBuilder
	configFile : configXML
Servlet配置：
	Class      : org.more.submit.support.web.TopFilter
	buildClass : org.more.submit.casing.more.WebMoreBuilder
	configFile : configXML
