submit系统v2.0的外壳提供程序。该外壳是使用spring作为action和过滤器。 在spring的配置文件中并不是所有的bean都是action。如果想使一个bean成为action需要像如下进行配置：
<bean name="mrmiAction" class="org.more.mrmi.JavaScriptSubmitManager">
<meta key="isAction" value="true" /><!-- submit使用 -->
</bean>
spring配置文件说明：
isAction：表示是否为一个action。
isFilter：表示是否为一个submit过滤器，submit过滤器除了需要实现ActionFilter接口之外还需要声明它。
isPublicFilter：表示是否为一个全局action过滤器。
