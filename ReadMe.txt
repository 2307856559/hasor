Hasor

    Hasor 是一款开源的 Java 应用开发框架。它是围绕 Guice 为核心创建的一系列模块组合而成。使用 Hasor 会加快软件软件开发效率并降低开发成本。目前 Hasor 由 Hasor-Core 、 Hasor-Web 、 Hasor-JDBC 三个主要模块组合而成。Hasor 的本质与Struts,Hibernate等单层框架不同，它可以提供一个以统一、高效的、友好的方式构造整个应用程序。并且可以将这些单层框架建立起一个连贯的体系，可以说Hasor是一个搭建开发环境的框架。
